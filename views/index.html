API projects: Image Search Abstraction Layer.

Description:

  • You can get the image URLs, alt text and page urls for a set of images relating to a given search string, using /search? followed by the desired query.
  • You can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • You can get a list of the most recently submitted search strings using /queries

Example usage:

https://mhl-searchimage.glitch.me/search?funny images
https://mhl-searchimage.glitch.me/queries
